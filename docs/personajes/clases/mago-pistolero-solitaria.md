# Orden del Arma Solitaria

**Orden de Mago Pistolero**

No todos los que nacieron con el don del mago pistolero tienen el privilegio de entrenar en una academia militar. Algunos son **vagabundos solitarios** que aprenden su oficio mediante el método de prueba y error, desarrollando habilidades y disparos rúnicos empleando puro instinto; otros dominan su técnica bajo la tutela de un mentor no afiliado.

---

## Rasgos de la Orden

### Nivel 1: Disparos Rúnicos de la Orden

Aprendes el disparo rúnico **Disparo con Truco**. Este disparo rúnico no cuenta para el número total de disparos rúnicos que puedes saber.

### Nivel 1: Tiro Salvaje

Has aprendido a mejorar tus ataques con una energía pura e impredecible.

Cuando realizas un ataque a distancia con un arma de fuego con la que tengas competencia, puedes **gastar un espacio de conjuro** de mago pistolero para mejorar la bala con pura energía mágica:

- El ataque inflige **1d8 de daño adicional por nivel** del espacio de conjuro utilizado
- Tira un d8 y consulta la tabla para determinar el tipo de daño

| d8 | Tipo de daño |
|----|--------------|
| 1 | Ácido |
| 2 | Frío |
| 3 | Fuego |
| 4 | Fuerza |
| 5 | Relámpago |
| 6 | Necrótico |
| 7 | Veneno |
| 8 | Trueno |

### Nivel 6: Ataque Adicional

Podrás realizar **dos ataques** en lugar de uno cuando lleves a cabo la acción de Atacar durante tu turno.

### Nivel 10: Unos Cuantos Disparos Más

Puedes potenciar tus disparos rúnicos a costa de tus otras capacidades arcanas.

Cuando gastas un espacio de conjuro de mago pistolero de **nivel 1 o superior** para recuperar usos del rasgo Disparos Rúnicos, tratas el espacio de conjuro como de **un nivel más alto**.

**Usos:** Una vez por descanso largo.

### Nivel 14: Improvisación Rúnica

Aprendes a improvisar rápidamente nuevos disparos rúnicos.

Cuando uses un disparo rúnico, puedes usar uno que **no conocieras antes** para resolver tu próximo ataque con disparo rúnico.

**Usos:** Una vez por descanso largo.
