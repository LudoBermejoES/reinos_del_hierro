# Guía del Mundo de los Reinos de Hierro

Este documento es una versión re-estructurada y formateada en Markdown del archivo de texto `ik-vol2-guiadelmundo.txt`, que es el resultado de un proceso de OCR sobre el PDF `ik-vol2-guiadelmundo_compress.pdf`.

Debido a la baja calidad del OCR, el texto puede contener errores y artefactos. El objetivo de este documento es presentar la información de una manera más legible y organizada.
